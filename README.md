# 🏥 Healthcare Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing healthcare data to generate meaningful insights using Excel and Power BI. The dashboard helps in understanding patient trends, medical conditions, hospital performance, and billing analysis.

---

## 🎯 Objectives
- Analyze patient demographics and medical conditions  
- Identify trends in hospital admissions  
- Evaluate hospital and insurance performance  
- Track billing and financial metrics  
- Support data-driven decision-making  

---

## 🛠️ Tools & Technologies
- Power BI  
- Microsoft Excel  
- Data Cleaning & Data Analysis  

---

## 📂 Dataset
Healthcare dataset containing:
- Patient details (Age, Gender, Blood Type)  
- Medical information (Condition, Medication, Test Results)  
- Hospital details (Hospital, Doctor, Admission Type)  
- Financial data (Billing Amount, Insurance Provider)  
- Dates (Admission & Discharge)  

---

## 📊 Dashboard Features

### 🔹 Page 1: Healthcare Overview
- Total Patients, Total Billing, Average Stay  
- Monthly patient trend  
- Gender distribution  
- Patients by medical condition  
- Interactive filters  

---

### 🔹 Page 2: Patient & Medical Analysis
- Patients by medical condition  
- Age group analysis  
- Gender vs condition analysis  
- Test results distribution  
- Admission type analysis  

---

### 🔹 Page 3: Financial & Hospital Analysis
- Billing by hospital  
- Billing by insurance provider  
- Average stay by hospital  
- Admission type vs billing  
- Monthly billing trend  

---

## 🔍 Key Insights
- Certain medical conditions like hypertension and diabetes are more common  
- Adult age group contributes the highest number of patients  
- Billing varies significantly across hospitals and insurance providers  
- Monthly trends show variation in patient admissions and billing  

---

## 💡 Business Impact
- Helps hospitals improve resource planning  
- Supports decision-making in healthcare management  
- Identifies high-cost areas for optimization  
- Improves understanding of patient trends
